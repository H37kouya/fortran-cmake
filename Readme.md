# Fortran Cmake Sample

## Setup

you must use cmake.

```shell script

$ brew install gcc

$ brew install cmake
``` 

## How to run

### First method

You have to install gcc(gfortran) and make, cmake for local environment.
```shell script
# コンパイルから実行までまとめて行う
$ make dev

# コンパイルを行う
$ make build

# コンパイル済みのコードの実行を行う
$ make run
```
